## 🎯 三分鐘快速測試

### 1️⃣ 創建測試比賽 (30秒)
1. 點擊「新增賽事」
2. 輸入比賽資訊：
   - 標題：測試比賽
   - 選手 A：費德勒
   - 選手 B：納達爾
   - 賽制：單盤決勝
3. 點擊「創建比賽」

### 2️⃣ 快速記分 (1分鐘)
1. 進入裁判台模式
2. 點擊「Ace」按鈕 3 次（費德勒得 3 分）
3. 點擊「雙誤」按鈕 2 次（納達爾得 2 分）
4. 交替點擊「致勝」和「失誤」按鈕各 3 次
5. 觀察統計數據自動累積

### 3️⃣ 測試新功能 (1.5分鐘)

#### 📸 比分截圖 (30秒)
1. 點擊底部「快速截圖」按鈕
2. 等待 2 秒自動生成
3. 點擊「下載圖片」保存
4. ✅ 完成！

#### 📊 進階分析 (30秒)
1. 切換到「紀錄」模式
2. 點擊頂部「進階」按鈕（紫色）
3. 切換「總覽」「發球分析」「動能分析」三個分頁
4. ✅ 完成！

#### 📄 PDF 報告 (30秒)
1. 在紀錄模式點擊「PDF」按鈕（紅色）
2. 在彈窗中點擊「生成 PDF」
3. 等待進度條完成
4. ✅ PDF 自動下載完成！

## 🆘 常見問題

### Q1: PDF 下載失敗？
**A**: 確認瀏覽器允許下載，檢查彈出視窗設定

### Q2: 截圖一片空白？
**A**: 等待 2-3 秒讓元素完全渲染

### Q3: 進階統計沒有數據？
**A**: 需要至少進行 5 分以上才會顯示走勢圖

### Q4: 複製圖片失敗？
**A**: 部分瀏覽器需 HTTPS，建議使用「下載」功能

### Q5: 計時器不準確？
**A**: 點擊暫停/恢復按鈕重新校準
